Backup & Restore

        
Clone this deployment into another project

      

      


        
Download Backup

        

          

            Downloads a full snapshot of this deployment — config, provider keys, channel tokens,
            approved users, chat history, memories, skills, and cron jobs — as a single .zip.
            Upload it via Restore below on another deployment to reproduce this bot there.
            The file is not encrypted — it contains
            your API keys and bot tokens in plain text, so store it somewhere safe.
          

          
        


        
Restore from Backup

        

          

            Uploading a backup here overwrites this
            deployment's current config, credentials, and data with the backup's contents, then
            briefly restarts the gateway. Anything changed here since the backup was taken (e.g.
            newer user approvals) can be lost. A safety snapshot of the current state is taken
            automatically first — see below if you need to recover it.

How to Setup

        
Get your Hermes Agent running in minutes

      

      


        
        

          

            1
            Get a Free LLM API Key
          

          
    
            
  1. 
              Go to openrouter.ai and create a free account.
            
    2. 
            
  2. 
              Once logged in, go to openrouter.ai/workspaces/default/keys and click Create Key.
              Copy the generated API key — it starts with sk-or-...
            
    3. 
            
  3. 
              OpenRouter offers many models, including free ones. To get started, you can pick a free model such as:
              nvidia/llama-3.3-nemotron-super-49b-v1:free
              Browse all available models at openrouter.ai/models — filter by "Free" to see no-cost options.
            
    4. 
            
  4. 
              Head over to the Setup page (left sidebar), select OpenRouter as provider, paste your API key, and enter the model name.
              
                Want a different provider (Anthropic, DeepSeek, Groq, Cerebras, Mistral, etc.)? Open the
                Hermes Dashboard from the sidebar — the
                Keys tab lists every provider Hermes supports out of the box, plus
                skills, toolsets, and analytics you can configure from there.
              
            
    5. 
          

        


        
        

          

            2
            Connect a Messaging Platform
          

          

            Hermes Agent can work through messaging platforms. Connecting one is optional — the easiest to set up is Telegram.
          

          
    
            
  1. 
              Open Telegram and search for @BotFather.
            
    2. 
            
  2. 
              Send /newbot and follow the prompts — give your bot a name and username.
            
    3. 
            
  3. 
              BotFather will reply with a Bot Token — it looks like 1234567890:ABCdefGhIjKlMnOpQrStUvWxYz.
              Copy this token.
            
    4. 
            
  4. 
              Go to the Setup page, enable Telegram under Messaging Channels, and paste the Bot Token.
            
    5. 
            
  5. 
              For Allowed User IDs, if you're just starting out, enter * to allow all users.
              You can restrict access later by specifying individual Telegram user IDs.
            
    6. 
            
  6. 
              Click Save & Start on the bottom right — your Hermes Agent is now live!
            
    7. 
          

        


        
        

          Facing an issue or something not working as expected? Open an issue on the official Hermes Agent repository.
